# Polymorphic - Project Overview

> **"The adaptable page builder"** — Lightweight. Powerful. Polymorphic.

## Vision Statement

Polymorphic is a modern WordPress page builder designed for agencies and professionals who demand speed without sacrificing power. Unlike bloated competitors, Polymorphic takes a minimalist approach: single postmeta storage, transient caching, and a clean React-based interface.

---

## Brand Identity

### Core Message
A builder that adapts to YOUR way of working, not the other way around.

### Key Differentiators
| Feature | Polymorphic | Elementor | Oxygen |
|---------|-------------|-----------|--------|
| Database footprint | Single postmeta | Multiple tables + revisions | Single postmeta |
| Bundle size | <50KB gzipped | >500KB | ~200KB |
| Learning curve | Low | Low | High |
| Developer-friendly | ✓ Hooks & filters | ✓ Limited | ✓ Full |
| React-based builder | ✓ Modern | Vue.js | Vanilla JS |

### Visual Language
- **Colors**: Dark mode primary, accent gradients matching flexibility/morphing
- **Typography**: Clean, modern sans-serif (Inter, Plus Jakarta Sans)
- **UI Style**: Glassmorphism subtle, depth via shadows, smooth animations
- **Icons**: Minimal, outlined style (Lucide/Phosphor)

## Competitive Positioning

                    POWER

         Oxygen ──────┼────── Polymorphic
                      │
    Gutenberg ────────┼────── Elementor

                   SIMPLICITY

### vs. Elementor
- **Our pitch**: "All the power, 10% of the bloat"
- Faster load times, cleaner database
- No upsell popups, no feature overload

### vs. Oxygen
- **Our pitch**: "Developer power, designer UX"
- Easier learning curve
- Better visual feedback
- Friendlier for teams

### vs. Gutenberg
- **Our pitch**: "Professional-grade page building"
- Full layout control
- Dedicated builder interface
- Advanced styling options
